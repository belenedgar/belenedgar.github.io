Dinosaur Chase


I created this game with the goal of integrating physical activity into a web game to encourage people to stay active while still playing video games.

How to play: The glasses on the screen will track your face, and you must move around the screen to avoid the dinosaurs hitting your glasses. If the glasses touch a dinosaur then it's game over. To get bonus points try to catch the floating egg with your glasses. Have fun!

This game was created to be used on a version of Scratch that Northwestern University's TIILT lab developed. To play, download the game file below, and load the file from your computer into this web browser.
